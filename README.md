# COMLAND demo

## Requirements
- MongoDB, Mongoose
- NodeJS

## Installation

### API (NestJS)
Install the dependencies and devDependencies and start the server.

```sh
cd api
npm i
npm run start
```

### Front-end (Angular)
Install the dependencies.

```sh
cd web
npm i
ng serve -o
```
